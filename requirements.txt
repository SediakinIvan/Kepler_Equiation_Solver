# Требования для проекта "Решатель уравнения Кеплера"
# Учебный проект по дисциплине Механика Космических полетов

## Компилятор и инструменты сборки
- Microsoft Visual Studio 2019 или выше
  - Platform Toolset: v143 (Visual Studio 2022)
  - Минимальная версия: v142 (Visual Studio 2019)
- MSBuild (входит в состав Visual Studio)
- Windows SDK версии 10.0 или выше

## Язык программирования
- Стандарт C++17 или выше (C++11 минимум)

## Стандартная библиотека
- Стандартная библиотека C++:
  - <iostream> - для ввода-вывода
  - <cmath> - для математических функций (sin, cos, sqrt, abs)

## Внешние зависимости
- Отсутствуют (используются только стандартные библиотеки C++)

## Платформа
- Операционная система: Windows 10 или выше
- Архитектура: x86, x64

## Опционально (для разработки)
- Система контроля версий: Git
- IDE: Microsoft Visual Studio (рекомендуется)

## Примечания
Этот проект является консольным приложением на C++ и не требует установки дополнительных библиотек или фреймворков. Все необходимые компоненты входят в состав стандартной библиотеки C++.
